ИНСТРУКЦИЯ ПО УСТАНОВКЕ.

Мы ценим Ваше время! Поэтому был разработан инсталляционный файл, с помощью которого установка HYIP-скрипта стала гораздо проще и быстрее.
Для этого необходимо сделать несколько действий:

1. Создайте базу данных на вашем сервере (Кодировка: windows-1251)

2. Пропишите все данные (в т.ч. и ключ) в файле cfg.php

3. Загрузите все файлы на хостинг

4. Запустите файл: httр://SITENAME.COM/install.php

5. В форме пропишите свои данные для администратора и нажмите кнопку ИНСТАЛЛИРОВАТЬ!

6. После удачной инсталляции удалите файл install.php

7. Дайте права на запись (777) файлу: rss.xml

8. Остальные настройки вы можете внести через панель администратора в меню "Настройки проекта"
